## 🚀 Currently Working On

## ⭐ Wino Mail

**Wino Mail** is a modern, native Windows mail client built with .NET 9 and UWP.

It is a community-driven continuation of the classic Windows Mail experience — designed to preserve the simplicity and familiarity users love while modernizing the architecture for performance and long-term sustainability.

### Key Highlights

- Native Windows UI (UWP / WinUI)
- IMAP / SMTP support
- Outlook & Gmail integrations
- Offline support
- Modern .NET 9 architecture
- Clean, responsive design (Light & Dark themes)
- Active open-source community

Wino Mail focuses on reliability, performance, and delivering a smooth desktop mail experience for power users.
